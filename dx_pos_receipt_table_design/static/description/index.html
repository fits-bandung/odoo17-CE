What is this module?

This module changes default odoo pos receipt design into table design

Features

    • Change pos receipt to table design
    • Show customer name if selected
    • Add receipt number on top to be more readable
Receipt example
1- Receipt number is current session
2- Show client name if selected
3- Discount of line
4- Receipt number is current session
5- Move served by to receipt bottom
